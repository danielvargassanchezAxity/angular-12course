Pipes

Un pipe sirve para transformar la informacion en como se visualizara en el DOM

Ejemplo de pipes:

DatePipe

{{ today | date: 'medium' }}

{{ today | date: 'short' }}

{{ today | date: 'long' }}

{{ today | date: 'full' }}

{{ today | date: 'shortDate' }}

{{ today | date: 'h:mm:ss a' }}

{{ today | date: 'dd/MM/yyyy' }}

Mas formatos

Uppercase y lower case

{{ 'uppercase pipe' | uppercase }}

{{ message | lowercase }}

Currency

{{ price | currency:'EUR':'symbol':'1.2-2' }}

{{ price | currency:'MXN':'symbol':'1.3-3' }}

{{ price | currency:'USD':'symbol':'1.2-2' }}
